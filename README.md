## 📋 Descripción

Sistema de gestión de biblioteca desarrollado con **JavaScript puro** como parte de mi aprendizaje para dominar el lenguaje antes de pasar a frameworks como React y Node.js. Este proyecto sigue prácticas profesionales de desarrollo: arquitectura modular, validaciones robustas, control de versiones con Git/GitHub y metodología SCRUM.

## 🎯 Objetivos de Aprendizaje

- ✅ Dominar JavaScript ES6+ (módulos, clases, arrow functions, etc.)
- ✅ Implementar validaciones profesionales
- ✅ Aplicar arquitectura modular y mantenible
- ✅ Usar Git y GitHub como un profesional
- ✅ Practicar metodología SCRUM (Sprints, Daily, Reviews)
- ✅ Escribir código limpio y autodocumentado
